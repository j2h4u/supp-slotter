# Quick Task 260506-kt5: Normalize trait ontology axes

## Goal

Normalize the trait ontology around clear current axes while removing unused or overlapping traits. Keep the planner simple: no new inference, only renamed/cleaned declarative traits.

## Tasks

1. Normalize trait identifiers and definitions.
   - Rename food-axis traits:
     - `intake:requires_food` -> `intake:food_required`
     - `intake:prefers_food` -> `intake:food_preferred`
     - `intake:prefers_empty_stomach` -> `intake:empty_preferred`
     - `intake:requires_fat_containing_meal` -> `intake:fat_meal_required`
   - Delete unused `intake:separate_from_food`, `intake:with_water_or_food`, and `risk:dose_monitoring`.
   - Rename `family:*_like` absorption traits to `competition:*_absorption`.
   - Delete unused `family:iron_like`.
   - Remove scheduling effects from `class:fat_soluble`.

2. Update references and planner namespace validation.
   - Update substance cards and tests to the new trait ids.
   - Replace registered namespace `family` with `competition`.
   - Preserve `effect:*` and `mechanism:*` ids.

3. Verify and document.
   - Regenerate `schedule.yaml`.
   - Update docs with the normalized axes.
   - Run `uv run planner.py check`, `uv run planner.py plan`, `uv run planner.py orphans`, and `uv run pytest`.
