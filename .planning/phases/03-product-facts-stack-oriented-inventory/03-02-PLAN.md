<task type="auto">
  <name>Task 2: Rewrite inventory schema and data to top-level stacks</name>
  <files>schema/inventory.schema.json, data/inventory.yaml</files>
  <action>
Rewrite `schema/inventory.schema.json` so the root object requires `version` and `stacks`.

`stacks` must require exactly:
- `daily`
- `training`
- `inactive`

Each stack is an object keyed by inventory item id. Each item must require `product` and may keep only inventory-owned fields:
- `product`
- `notes`
- `traits_override`

Do not allow per-item `stack`, `brand`, or `dose`.

Rewrite `data/inventory.yaml` into:

```yaml
version: 1
stacks:
  daily:
    item_id:
      product: product_id
  training:
    item_id:
      product: product_id
  inactive:
    item_id:
      product: product_id
```

Keep existing inventory item ids and product refs. Keep inventory-specific `traits_override` and operator-specific notes. Product facts were copied in Plan 03-01; do not preserve them in inventory.
  </action>
