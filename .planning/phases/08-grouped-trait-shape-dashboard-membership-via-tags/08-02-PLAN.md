# Plan 08-02: Stage 2a — Documentation (DT-11)

## Context

Stage 2 plans are independent additive commits landing after Stage 1 (08-01). This plan updates the two reference docs and README for the grouped model.

**Important grep hygiene (per cross-AI review):** `grep 'traits:'` matches `from_traits:` as a substring — false positives. All stale-form checks in this plan use anchored regex patterns: `^\s*traits:` (line-start whitespace, then `traits:` as a YAML key) for the flat-form check, and `^\s*taking:` for the old dashboard mechanism. The valid `from_traits:` key is not matched by these anchored patterns.
