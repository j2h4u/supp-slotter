       "from_traits: { is: [antioxidant] } is intensional — open-world: any future substance that acquires is: antioxidant automatically joins the cluster without editor intervention. from_traits: { dashboard: [foo] } is extensional — closed curation: only substances explicitly tagged dashboard: foo are members. Mitigation: operator hygiene + occasional review via review-substance output. Real-world frequency expected to be low given ~50 substance cards and rare renames."
